Principais Formatações

Negrito ou Destaque

Nesta frase temos um TERMO EM NEGRITO usando a tag <b>, que é uma tag NÃO SEMÂNTICA.

Nesta frase temos um TERMO EM DESTAQUE usando a tag <strong> , que é uma tag SEMÂNTICA.

Itálico ou Ênfase

Nesta frase temos um TERMO EM ITÁLICO usando a tag <i> , que é uma tag NÃO SEMÂNTICA.

Nesta frase temos um TERMO EM ÊNFASE usando a tag <em> , que é uma tag SEMÂNTICA.

Texto Marcado

Podemos usar um TEXTO MARCADO usando tag <mark> , que é uma tag SEMÂNTICA.

Texto Grande ou Pequeno

Nesta frase, temos um texto grande e um texto pequeno , onde <big> é uma tag NÃO SEMÂNTICA e <small> é uma tag SEMÂNTICA.

Texto Deletado

Podemos marcar um texto como deletado (também chamado excluído ou riscado) para indicar que ele pode ser lido, porém, não deve ser considerado. A tag <del> é uma tag SEMÂNTICA.

Texto Inserido ou Sublinhado

Podemos marcar um TERMO COMO INSERIDO para dar uma noção de acréscimo em seu conteúdo e indicar que ele foi adicionado posteriormente. A tag <ins> é uma tag SEMÂNTICA.

Nesta frase temos um TERMO SUBLINHADO que utiliza a tag <u> , sendo uma tag NÃO SEMÂNTICA.

Texto Sobrescrito

Podemos utilizar este tipo de formatação para inserir coisas do tipo xy+3, utilizando a tag <sup>..

Texto Subescrito

Podemos utilizar este tipo de formatação para inserir coisas do tipo Log28, utilizando a tag <sub>.
